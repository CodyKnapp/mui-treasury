This feature works only for CollapsibleSidebar

- [Permanent EdgeSidebar](/layout/api-reference/edge-sidebar-builder#registerpermanentconfig)
- [Persistent EdgeSidebar](/layout/api-reference/edge-sidebar-builder#registerpersistentconfig)

#### What's it looks like

When your sidebar collapsed and user hovered on the sidebar (with auto-expanded enabled), the sidebar
will automatically expand with the same width as uncollapsed state.

Take a look at [Gmail clone](/layout/clones/gmail?bgColor=ff4a3e&accent=000000&dark=true) and hover on the main sidebar.

#### Config

To enable this feature, add `autoExpanded: true` in configuration (it is `false` by default)

```js
scheme.configureEdgeSidebar(builder => {
  builder
    .create('primarySidebar', { anchor: 'left' })
    .registerPermanentConfig('md', {
      width: 256,
      collapsible: true,
      collapsedWidth: 64,
      autoExpanded: true,
    });
});
```
