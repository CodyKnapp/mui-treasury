A component for trigger `open` state of Temporary and Persistent of EdgeSidebar.
This component will be hidden based on the configuration of EdgeSidebar, so you don't need to
assign any props to it except sidebarId that it need to know which sidebar should update.

## Usage example

```jsx
import styled from 'styled-components'
import Layout, { getLayoutComponents } from '@mui-treasury/layout';

const { Root, DrawerSidebar, SidebarTrigger } = getLayoutComponents(styled)

cosnt scheme = Layout()

scheme.configureEdgeSidebar((builder) => {
  builder
    .create("primarySidebar", { anchor: "left" })
    .registerPersistentConfig("md", {
      width: 256,
      collapsible: false,
      persistentBehavior: "fit",
    });
});

const App = () => {
  return (
    <Root scheme={scheme}>
      <SidebarTrigger sidebarId="primarySidebar" />
      <DrawerSidebar sidebarId="primarySidebar">
        sidebar content
      </DrawerSidebar>
    </Root>
  )
}
```

## Props

- `sidebarId`
  - **type** : string
  - **required** : yes
  - **description** : sidebarId that will be updated. If this component renders inside of EdgeSidebar you don't need to specify `sidebarId`
    unless you want to target to other sidebar.
    ```jsx
      const App = () => {
        return (
          <Root scheme={scheme}>
            <DrawerSidebar sidebarId="primarySidebar">
              <SidebarTrigger /> // will trigger primarySidebar
              sidebar content
            </DrawerSidebar>
          </Root>
        )
      }
    ```
- `SvgIconProps`
  - **type** : object
  - **description** : props that are sent to [SvgIcon](https://material-ui.com/api/svg-icon/#props)
- [other props](https://material-ui.com/api/icon-button/#props) refer to Material-UI IconButton

## Styling

SidebarTrigger use `IconButton` that contains `SvgIcon` insider. All of the props that you provide will go to `IconButton` except SvgIconProps will send to `SvgIcon`

- [IconButton](https://material-ui.com/api/icon-button/#props)
- [SvgIcon](https://material-ui.com/api/svg-icon/#props)

```jsx
<SidebarTrigger
  classes={{ root: 'custom-trigger' }}
  SvgIconProps={{
    className: 'custom-icon',
  }}
/>
```
