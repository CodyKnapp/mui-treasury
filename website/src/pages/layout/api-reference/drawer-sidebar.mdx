Drawer sidebar will look for edge sidebar config and render based on them.

## Usage example

```jsx
import styled from 'styled-components'
import Layout, { getLayoutComponents } from '@mui-treasury/layout';

const { Root, DrawerSidebar } = getLayoutComponents(styled)

cosnt scheme = Layout()

scheme.configureEdgeSidebar((builder) => {
  builder
    .create("primarySidebar", { anchor: "left" })
    .registerPersistentConfig("md", {
      width: 256,
      collapsible: false,
      persistentBehavior: "fit",
    });
});

const App = () => {
  return (
    <Root scheme={scheme}>
      <DrawerSidebar sidebarId="primarySidebar">
        sidebar content
      </DrawerSidebar>
    </Root>
  )
}
```

You can render DrawerSidebar at any order below `Root`, no need to concern about anchor left or right.

## Props

- `sidebarId`
  - **type** : string
  - **required** : yes
  - **description** : id that will match sidebar config
- [other props](https://material-ui.com/api/drawer/#props) excluding `anchor` and `variant`

## Styling

Because drawer sidebar use `Drawer` from material-ui, please read [official doc](https://material-ui.com/api/drawer/#css)
