This component is a combination of `Footer`, `Container` and `InsetAvoidingView` without extra functionality.

## Usage example

```jsx
import styled from 'styled-components'
import Container from '@material-ui/core/Container';
import Layout, { getLayoutComponents } from '@mui-treasury/layout';

const {
  Root,
  InsetFooter,
  InsetAvoidingView,
} = getLayoutComponents(styled)

cosnt scheme = Layout()

const App = () => {
  return (
    <Root scheme={scheme}>
      <InsetFooter>
        footer
      </InsetFooter>
      // same as
      <Footer>
        <Container>
          <InsetAvoidingView>
            footer
          </InsetAvoidingView>
        </Container>
      </Footer>
    </Root>
  )
}
```

## Props

- `ContainerProps`
  - **type** : object
  - **description** : props that pass to [Container](https://material-ui.com/api/container/#container-api)
- `InsetAvoidingViewProps`
  - **type** : object
  - **description** : props that pass to [InsetAvoidingView](/layout/api-reference/inset-avoiding-view)
- Other props will be passed to [Footer](/layout/api-reference/footer)

## Styling

pass `className` inside `ContainerProps` or `InsetAvoidingViewProps`

```jsx
<InsetFooter
  ContainerProps={{
    className: 'custom-container',
    // or use classes
  }}
  InsetAvoidingViewProps={{
    className: 'custom-view',
  }}
>
  footer
</InsetFooter>
```
