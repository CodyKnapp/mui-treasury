This component is a simple `<div />` that has these style. Mostly, for AbsoluteInsetSidebar layout like [messenger](https://messenger.com).

```jsx
{
  height: "100vh",
  display: "flex",
  flexDirection: "column",
  overflow: "hidden",
}
```

## Usage example

```jsx
import styled from 'styled-components'
import Layout, { Root, Fullscreen } from '@mui-treasury/layout';

cosnt scheme = Layout()

const App = () => {
  return (
    <Fullscreen>
      <Root scheme={scheme}>
        <Footer>footer</Footer>
      </Root>
    </Fullscreen>
  )
}
```

## Props

any props will be sent to `<div />`

## Styling

use `className`
