Use this component when you want to avoid Fixed or Absolute InsetSidebar. Mostly, this component is used in Footer section
in combination with `Footer` and `Container`.

## Usage example

```jsx
import styled from 'styled-components'
import Container from '@material-ui/core/Container';
import Layout, { getLayoutComponents } from '@mui-treasury/layout';

const {
  Root,
  Footer,
  InsetAvoidingView,
} = getLayoutComponents(styled)

cosnt scheme = Layout()

const App = () => {
  return (
    <Root scheme={scheme}>
      <Footer>
        <Container>
          <InsetAvoidingView>
            footer
          </InsetAvoidingView>
        </Container>
      </Footer>
    </Root>
  )
}
```

## Props

all of props will pass to `div`

## Styling

use `className`
