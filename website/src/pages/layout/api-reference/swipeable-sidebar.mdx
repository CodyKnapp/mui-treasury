Swipeable sidebar will look for edge sidebar config and render based on them. Everything is the same as
[Drawer Sidebar](/layout/api-reference/drawer-sidebar) except extra props that [Material-ui Swipeable Drawer](https://material-ui.com/api/swipeable-drawer/#swipeabledrawer-api) accepts

## Usage example

```jsx
import styled from 'styled-components'
import Layout, { getLayoutComponents } from '@mui-treasury/layout';

const { Root, SwipeableSidebar } = getLayoutComponents(styled)

cosnt scheme = Layout()

scheme.configureEdgeSidebar((builder) => {
  builder
    .create("primarySidebar", { anchor: "left" })
    .registerTemporaryConfig("xs", {
      width: 'auto',
    })
    .registerPersistentConfig("md", {
      width: 256,
      collapsible: false,
      persistentBehavior: "fit",
    });
});

const App = () => {
  return (
    <Root scheme={scheme}>
      <SwipeableSidebar sidebarId="primarySidebar">
        sidebar content
      </DrawerSidebar>
    </Root>
  )
}
```

## Props

- `sidebarId`
  - **type** : string
  - **required** : yes
  - **description** : id that will match sidebar config
- [other props](https://material-ui.com/api/swipeable-drawer/#swipeabledrawer-api) excluding `anchor` and `variant`
