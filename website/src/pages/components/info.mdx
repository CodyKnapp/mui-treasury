One of the component that appears in any web app. It consists of `Title`, `Subtitle` and sometimes `Caption`.
It looks like this,

import {
  Info,
  InfoTitle,
  InfoSubtitle,
  InfoCaption,
} from '@mui-treasury/components/info';

<SimpleShowcase>
  <Info>
    <InfoTitle>This is Title</InfoTitle>
    <InfoSubtitle>Details or Tagline usually appears here!</InfoSubtitle>
  </Info>
</SimpleShowcase>

It frequently appears in `ListItem` with `Avatar`.

import { Row } from '@mui-treasury/components/flex';
import Avatar from '@material-ui/core/Avatar';

<SimpleShowcase>
  <Row gap={2} display="inline-flex" textAlign="left" bgcolor="#fff">
    <Avatar variant="rounded" src="https://i.pravatar.cc/300" />
    <Info>
      <InfoTitle variant="subtitle1">Maria Iscelca</InfoTitle>
      <InfoSubtitle variant="subtitle2">Can you please send me more detail?</InfoSubtitle>
      <InfoCaption>10.45 AM</InfoCaption>
    </Info>
  </Row>
</SimpleShowcase>
